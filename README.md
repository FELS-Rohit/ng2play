# ng2play

The source code for running this projects comes in both ES6 and TypeScript, so we can explore both options separately

## Requirements
```
npm install -g http-server
npm install
bower install
```

##ES6
Located in `src/`

``` 
npm install
bower install
http-server
```

browse to `http://localhost:8080`

##TypeScript
Located in `src-ts/`

``` 
gulp ts-compile
http-server -p 8081
```

browse to `http://localhost:8081`
